Assets structure for Sheikh Cosmetics website

Use these folders to replace website visuals with your own files.

Main folders

- `slider/`
  Put homepage slider images here.

- `logos/`
  Put reusable brand logos here.

- `brands/`
  Put brand hero images or brand-specific banners here.

- `products/`
  Put product images here, grouped by brand.

- `videos/`
  Put video poster images or local video assets here if needed later.
